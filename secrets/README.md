# secrets/

This directory holds OAuth credentials for the YouTube uploader. **Nothing
in this directory is committed to git** (see the repo `.gitignore`).

## Files that live here

| File | Origin | Purpose |
|---|---|---|
| `client_secret.json`   | Downloaded from Google Cloud Console | OAuth client identity (the "app") |
| `youtube_token.json`   | Generated by `scripts/youtube_auth.py` | Refresh + access tokens for the channel |

## One-time setup for @BanksytheQuant uploads

1. Go to https://console.cloud.google.com and create a new project (or use an existing one).
2. **APIs & Services → Library** → enable **YouTube Data API v3**.
3. **APIs & Services → OAuth consent screen**:
   - User type: **External**
   - Fill the required fields (app name, your email).
   - **Scopes**: add `.../auth/youtube.upload`
   - **Test users**: add the Google account that owns the @BanksytheQuant YouTube channel.
4. **APIs & Services → Credentials → Create Credentials → OAuth client ID**:
   - Application type: **Desktop app**
   - Click **Download JSON**, rename the file to `client_secret.json`, and drop it into this `secrets/` directory.
5. From the repo root, run:
   ```
   python scripts/youtube_auth.py
   ```
6. Your browser will open. Sign in as the Google account that owns @BanksytheQuant.
7. Google may warn that the app isn't verified — that's expected for a personal-use app in **test mode**. Click **Advanced** → **Go to (App Name) (unsafe)** → grant the requested permission.
8. The script writes `youtube_token.json` here. Done.

After step 8, the Gradio Podcast Studio's **Upload to YouTube** section will work end-to-end. The refresh token is long-lived; you only need to redo this if you revoke access or rotate the channel's Google credentials.

## Quota

`videos.insert` costs **1,600 units** per upload. Default daily quota: **10,000 units**, so **~6 uploads per day per Google Cloud project**. If you need more, request a quota increase from the YouTube Data API v3 page in the Cloud Console (free, but takes a few business days).

## Privacy default

Uploads default to **private** (only you can see them). The Podcast Studio UI lets you switch to **unlisted** (anyone with the link) or **public** before clicking Upload.
